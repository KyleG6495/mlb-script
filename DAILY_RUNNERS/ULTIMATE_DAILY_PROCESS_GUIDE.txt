🔧 **TROUBLESHOOTING GUIDE**
============================

**Problem: "FanDuel slate not found"**
✅ Solution: Ensure `fd_slate_today.csv` exists and has correct format

**Problem: "No enhanced data found"** 
✅ Solution: Run `1_DATA_PIPELINE.bat` first

**Problem: "Player ID mismatch errors"**
✅ Solution: Our FanDuel-first architecture prevents this! Start with FanDuel slate.

**Problem: "Low projections (under 130 FPPG)"**
✅ Solution: 
- Check weather data loaded correctly
- Verify ML projections are recent
- Manual spot-check top players

**Problem: "Salary too low (under $30K)"**
✅ Solution: Algorithm is being too conservative - manually upgrade 1-2 plays

**Problem: "Upload fails to FanDuel"**
✅ Solution: 
- Check file format (9 columns: P,C/1B,2B,3B,SS,OF,OF.1,OF.2,UTIL)
- Verify player IDs match FanDuel format (######-####)
